# up01 db day 7

Решение практических работ 23-27 по PHP, MySQL, mysqli и PDO.

## Структура

- `index.html` - точка входа с ссылками на все части.
- `showList.php` - список товаров через `mysqli`, категории и пагинация.
- `showTable.php` - таблица через `mysqli`, сортировка, фильтрация и кнопки изменения.
- `editTable.php` - формы добавления, редактирования и подтверждения удаления.
- `saveTable.php` - сохранение изменений через `mysqli`.
- `index.php` - витрина товаров через `PDO`.
- `info.php` - подробная страница товара, фото и добавление в корзину через `PDO`.
- `header.php`, `footer.php`, `connection.php` - общие модули.
- `games.sql` - база `market`, таблицы `games`, `photos`, `cart` и тестовые данные.

## Запуск

1. Скопируй папку `up01_db` в папку сайта OpenServer.
2. Импортируй `games.sql` в MySQL.
3. Проверь настройки в `connection.php`:
   - `DB_HOST`
   - `DB_NAME`
   - `DB_USER`
   - `DB_PASSWORD`
4. Открой `index.html` в браузере через локальный сервер.

Код разбит на небольшие логические блоки, комментарии короткие и с маленькой буквы.
